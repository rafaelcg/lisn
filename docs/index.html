Built for focused capture.
Designed for real release.

⚡

Tray-First

Start from the menu bar, keep the interface compact, and drop into the full dashboard only when you need it.

🎯

App + Window Capture

Pick a target app or window instead of recording your whole machine. The source list filters out system noise.

☁️

Cloud Transcription

Lisn sends the finished recording through its transcription relay after capture stops, so the app works out of the box.

💾

Session History

Every capture is stored in local SQLite history so you can reopen, inspect, and export it later.

📄

Export Cleanly

Ship transcripts as Markdown or plain text for meeting notes, research logs, support calls, and internal docs.

🔓

Open Source

The app is MIT licensed and the full codebase is public, including release automation and the hosted landing page.

Download Lisn

Install the signed macOS alpha, open Screen Recording permissions once, and start capturing.
